# M7Б. Моделирование идеального газа в закрытом сосуде

Комплексная симуляция движения большого числа молекул в закрытом сосуде с учётом статистической физики и термодинамики. Программа исследует переход к равновесному состоянию, флуктуации термодинамических величин и энтропийные процессы.

## Описание проекта

Программа реализует молекулярно-кинетическую модель идеального газа, позволяющую изучать статистические закономерности в системах с большим числом частиц. Система моделирует упругие столкновения молекул между собой и со стенками сосуда, отслеживает эволюцию к равновесию и анализирует термодинамические флуктуации.

## Математическая модель

### Уравнения движения

Каждая молекула моделируется как твёрдый шарик с массой $m$ и радиусом $r$. Движение описывается уравнениями Ньютона:

$$m\frac{d^2\vec{r}_i}{dt^2} = \vec{F}_i$$

где для идеального газа в отсутствие внешних полей $\vec{F}_i = 0$, за исключением моментов столкновений.

### Моделирование столкновений

**Упругое столкновение молекул:**

При столкновении двух молекул $i$ и $j$ выполняются законы сохранения импульса и энергии:

$$\vec{v}_i' = \vec{v}_i - \frac{2m_j}{m_i + m_j} \frac{(\vec{v}_i - \vec{v}_j) \cdot (\vec{r}_i - \vec{r}_j)}{|\vec{r}_i - \vec{r}_j|^2} (\vec{r}_i - \vec{r}_j)$$

$$\vec{v}_j' = \vec{v}_j - \frac{2m_i}{m_i + m_j} \frac{(\vec{v}_j - \vec{v}_i) \cdot (\vec{r}_j - \vec{r}_i)}{|\vec{r}_j - \vec{r}_i|^2} (\vec{r}_j - \vec{r}_i)$$

**Столкновение со стенками:**

При упругом столкновении со стенкой компонента скорости, нормальная к стенке, меняет знак:

$$\vec{v}' = \vec{v} - 2(\vec{v} \cdot \vec{n})\vec{n}$$

где $\vec{n}$ - нормаль к поверхности стенки.

### Термодинамические величины

**Температура:** (из средней кинетической энергии)
$$T = \frac{2}{3k_B} \langle E_k \rangle = \frac{m}{3k_B} \langle v^2 \rangle$$

**Давление:** (из импульса, передаваемого стенкам)
$$P = \frac{1}{3} \frac{N}{V} m \langle v^2 \rangle = \frac{N k_B T}{V}$$

**Энтропия идеального газа:**

$$S = N k_B \left[ \ln\left(\frac{V}{N}\left(\frac{4\pi m E}{3N h^2}\right)^{3/2}\right) + \frac{5}{2} \right]$$

### Флуктуации

**Флуктуации числа частиц:**
$$\langle (\Delta N)^2 \rangle = N$$

**Флуктуации энергии:**
$$\langle (\Delta E)^2 \rangle = k_B T^2 C_V$$

## Сценарии моделирования

### 1. Переход к равновесному состоянию

Исследуется эволюция системы из начального неравновесного состояния:

- Начальное разделение сосуда на две половины
- Различные начальные условия в половинах (число частиц, температура)
- Мониторинг выравнивания температур и концентраций
- Расчёт изменения энтропии системы

**Параметры по умолчанию:**
- Общее число частиц: 500
- Объём сосуда: 2.0 м³
- Время моделирования: 3.0 с
- Шаг по времени: 0.001 с

### 2. Исследование флуктуаций

Анализ статистических флуктуаций после установления равновесия:

- Флуктуации числа частиц в половинах сосуда
- Флуктуации температуры
- Сравнение с теоретическими предсказаниями
- Исследование зависимости от числа частиц

**Особый режим:**
- Малое число частиц (N = 10)
- Усиленные относительные флуктуации
- Проверка выполнения термодинамических законов

### 3. Проверка распределений

Верификация соответствия равновесных распределений теоретическим:

- Распределение Максвелла по скоростям
- Распределение Больцмана в поле тяжести
- Проверка уравнения состояния идеального газа

## Примеры результатов

### Распределение Максвелла

При установлении равновесия распределение скоростей частиц точно следует теоретической кривой Максвелла-Больцмана для двумерного случая:

$$f(v) = \frac{m}{kT} v e^{-mv^2/2kT}$$

### Барометрическая формула

В поле тяжести распределение частиц по высоте соответствует барометрической формуле:

$$n(h) = n_0 e^{-mgh/kT}$$

### Флуктуации числа частиц

Для системы с $N$ частиц среднеквадратичная флуктуация числа частиц в половине сосуда:

$$\frac{\sqrt{\langle (\Delta N)^2 \rangle}}{N} = \frac{1}{\sqrt{N}}$$

что подтверждается численными экспериментами.

### Энтропийные процессы

При переходе к равновесию энтропия системы монотонно возрастает, демонстрируя выполнение второго начала термодинамики.

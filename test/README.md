# Аналитические тесты физической модели бильярда

## Описание тестов

Набор unit-тестов для проверки корректности физической модели столкновений бильярдных шаров. Все тесты основаны на аналитических решениях с известными результатами.

## Аналитические выкладки

### 1. Энергия одного шара
**Формула:** \( E = \frac{1}{2}mv^2 \)
**Проверка:** Расчет кинетической энергии для изолированного шара

### 2. Упругое столкновение со стенкой  
**Условие:** Бесконечно тяжелая стенка
**Решение:** \( v' = -v \), \( E' = E \)
**Проверка:** Полное сохранение энергии

### 3. Лобовое упругое столкновение одинаковых масс
**Условие:** \( m_1 = m_2 \), \( \vec{v}_1 \parallel \vec{v}_2 \)
**Решение:** \( \vec{v}_1' = \vec{v}_2 \), \( \vec{v}_2' = \vec{v}_1 \)
**Проверка:** Обмен скоростями

### 4. Сохранение энергии и импульса
**Законы:** 
- \( \sum m_i\vec{v}_i = const \)
- \( \sum \frac{1}{2}m_iv_i^2 = const \)
**Проверка:** Точное сохранение в упругих столкновениях

### 5. Касательное столкновение
**Условие:** \( \vec{v}_1 \perp \vec{r}_{12} \)
**Решение:** Минимальное взаимодействие
**Проверка:** Скорости почти не изменяются

### 6. Законы Гука и Герца
**Формулы:**
- Гук: \( F = k\Delta x \)
- Герц: \( F = k\Delta x^{3/2} \)
**Проверка:** Корректность расчета сил деформации

# M1. Основная задача электростатики

Моделирование электростатического поля между двумя металлическими электродами методом моментов. Программа находит распределение заряда на поверхностях электродов, вычисляет электрическое поле в пространстве и ёмкость системы.

## Описание проекта

Программа реализует численный метод моментов (Method of Moments, MoM) для решения задачи электростатики. Поверхности проводников разбиваются на малые плоские элементы (триангуляция), для каждого из которых находится заряд $q_i$, обеспечивающий заданный потенциал на электродах. По найденному распределению заряда строятся силовые линии поля и вычисляется ёмкость системы.

## Математическая модель

### Постановка задачи

Два металлических электрода поддерживаются при постоянной разности потенциалов:

$$\Delta\varphi = V$$

Полные заряды электродов противоположны и равны по модулю: $Q_1 = -Q_2 = Q$.

### Метод моментов

Поверхность каждого проводника разбивается на $N$ малых плоских элементов. Каждый элемент считается точечным зарядом $q_i$, расположенным в его центре. Потенциал в центре $i$-го элемента, создаваемый всеми остальными зарядами:

$$\varphi_i = \sum_{j \neq i} \frac{q_j}{4\pi\varepsilon_0 \, r_{ij}}$$

где $r_{ij} = |\vec{r}_i - \vec{r}_j|$ — расстояние между центрами элементов $i$ и $j$.

### Система линейных уравнений

Условие равенства потенциала каждого элемента потенциалу соответствующего проводника $\varphi_i^{(0)}$ записывается в матричном виде:

$$\mathbf{A} \, \vec{q} = \vec{\varphi}$$

где матрица коэффициентов влияния:

$$A_{ij} = \begin{cases} \dfrac{1}{4\pi\varepsilon_0 \, r_{ij}}, & i \neq j \\[6pt] \dfrac{1}{4\pi\varepsilon_0 \, r_{\text{self},i}}, & i = j \end{cases}$$

Диагональный элемент $A_{ii}$ учитывает собственный потенциал элемента (самовлияние); для квадратного элемента со стороной $a$ он приближённо равен:

$$A_{ii} \approx \frac{1}{4\pi\varepsilon_0} \cdot \frac{1}{0.4517 \, \sqrt{S_i}}$$

где $S_i$ — площадь элемента. Система решается стандартными методами линейной алгебры:

$$\vec{q} = \mathbf{A}^{-1} \vec{\varphi}$$

### Электрическое поле

По найденным зарядам $q_i$ вектор напряжённости электрического поля в произвольной точке $\vec{r}$:

$$\vec{E}(\vec{r}) = \frac{1}{4\pi\varepsilon_0} \sum_{i=1}^{N} q_i \, \frac{\vec{r} - \vec{r}_i}{|\vec{r} - \vec{r}_i|^3}$$

### Ёмкость системы

Полный заряд одного электрода:

$$Q = \sum_{i \in \text{electrode}_1} q_i$$

Ёмкость конденсатора:

$$C = \frac{Q}{\Delta\varphi} = \frac{Q}{V}$$

## Алгоритм

1. **Дискретизация поверхностей** — триангуляция или разбиение на прямоугольные элементы; вычисление центров и площадей элементов.
2. **Формирование матрицы** $\mathbf{A}$ — заполнение коэффициентов взаимного и собственного влияния.
3. **Задание вектора потенциалов** $\vec{\varphi}$ — элементы первого электрода имеют потенциал $+V/2$, второго — $-V/2$.
4. **Решение СЛАУ** — `numpy.linalg.solve` или `scipy.linalg.solve`.
5. **Расчёт поля** $\vec{E}(\vec{r})$ в узлах расчётной сетки.
6. **Визуализация** — построение картины силовых линий и эквипотенциальных поверхностей.
7. **Расчёт ёмкости** — суммирование зарядов одного электрода.

## Параметры по умолчанию

| Параметр | Значение |
|---|---|
| Разность потенциалов $V$ | 1 В |
| Число элементов на электрод | 100–500 |
| Электрическая постоянная $\varepsilon_0$ | $8.854 \times 10^{-12}$ Ф/м |
| Метод решения СЛАУ | `scipy.linalg.solve` |

## Рассматриваемые конфигурации

### 1. Две концентрические сферы

Точное значение ёмкости для верификации:

$$C_{\text{exact}} = 4\pi\varepsilon_0 \frac{R_1 R_2}{R_2 - R_1}$$

### 2. Две раздельные сферы

Приближённая формула (для $d \gg R$):

$$C \approx 2\pi\varepsilon_0 \left(\frac{1}{R_1} + \frac{1}{d} \right)^{-1}$$

### 3. Плоский конденсатор (две пластины конечных размеров)

Приближение бесконечных пластин (для верификации):

$$C_{\text{plate}} = \varepsilon_0 \frac{S}{d}$$

где $S$ — площадь пластины, $d$ — расстояние между ними.
